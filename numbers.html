Interesting numbers in radiology

Often in medicine we make decisions based on numbers. It can be daunting to encounter numerical data without an intuitive understanding of what the numbers mean. For each type of quantity we are interested in, it's useful to have a reference point in memory for establishing context or doing back of the envelope calculations. These standard values are frequently passed down from attending to resident in read-out sessions without reference to how they were obtained. I've tried to list a few landmark values I've found useful, each with a reference.
